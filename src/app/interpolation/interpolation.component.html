Interpolation

What you can do

Interpolation: "{{ name }}" is a class property with from variable: name

2 + 2 = {{2 + 2}}

{{"Welcome" + name}}

{{"Length of name: " + name.length + " letters"}}

{{"Name in upper case: " + name.toUpperCase()}}

{{greetName()}}


What you cannot do!

What you cannot do is bind a property (or assign it) to an expression with in Curle Braces, like "a = 1 + 3"

And you cannot access the global variables like Window . Do do this, you have to write a function in ts. This is the window location href: {{windowURL}}.
